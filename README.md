## Esercitazione 3

### Prenotazioni e Time Slot

1) Studiare le API delle classi java.util.Calendar e java.util.GregorianCalendar
2) Clonare il repository nel proprio IDE. 
Si devono implementare tutti i metodi contrassegnati con "// TODO" nelle classi:

- TimeSlot.java
- Prenotazione.java

Nelle classi sono specificate le API tramite commenti javadoc. 
Sono presenti test JUnit per entrambe le classi. Si faccia particolare 
attenzione alla ridefinizione dei metodi equals, hashCode e compareTo 
come studiato a lezione.
